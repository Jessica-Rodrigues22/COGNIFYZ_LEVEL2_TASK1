# Cognifyz Level 2 - Task 1

## Project Overview
This project demonstrates the use of the Bootstrap framework to create a responsive card component and a responsive grid layout. The design automatically adapts to different screen sizes, providing a smooth user experience on desktop, tablet, and mobile devices.

## Features
- Responsive Card Component
- Responsive Grid Layout
- Mobile-Friendly Design
- Bootstrap 5 Framework
- Clean and Modern User Interface

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
